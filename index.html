Privacy Policy

Last Updated: January 11, 2026 (v1.2.1)

TabControl collects no personal data, tracking information, or user content.

Permissions Used

Note for v1.2.1+: The extension no longer requests the tabs permission. It performs tab switching using basic API capabilities (activeTab) that do not require access to your browsing history, tab titles, or URLs.

  • storage: Store user hotkey preferences locally on the device

What TabControl Does

  • Enables fullscreen tab switching via configurable hotkeys (e.g., Ctrl+Shift+Left/Right)
  • Shows temporary hotbar overlay (arrows) at screen bottom during navigation
  • Works in fullscreen/F11 mode across Chrome and Opera browsers

No Data Collection

  • ✅ No network requests to external servers
  • ✅ No user browsing history access
  • ✅ No personal information stored beyond local hotkey settings
  • ✅ No analytics, tracking pixels, or third-party services

Local Storage

User-configured hotkeys stored in browser localStorage only. Data stays on device, cleared if extension uninstalled. No sync to servers.

Chrome Web Store Privacy Label

"No data collected"
